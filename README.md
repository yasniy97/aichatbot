How to run.
1. Copy the version of your desired OS into your local directory.
2. Get the Ollama API key by signing up or login to Ollama website: https://ollama.com or;
3. Get your OpenAI API Key from https://platform.openai.com
4. Open your teminal.
5. Ollama : Export the API key by keying this command.
6.     Linux/MacOs : export OLLAMA_API_KEY=your_api_key_here
7.     Windows : set OLLAMA_API_KEY=your_api_key_here
8. OpenAI : Export the API key by keying this command on your terminal.
9.     Linux : export OPENAI_API_KEY=your_api_key_here
10.    Windows : set OPENAI_API_KEY=your_api_key_here
11. run the program i.e pmchat-windows-amd64.exe (windows) or ./pmchat-linux-amd64 or;
12.         pmchat-openai-windows-amd64.exe
