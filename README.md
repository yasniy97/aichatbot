How to run.
1. Copy the version of your desired OS into your local directory.
2. Get the Ollama API key by signing up or login to Ollama website.
3. Open your teminal.
4. Export the API key by keying this command.
5.     Linux/MacOs : export OLLAMA_API_KEY=your_api_key_here
6.     Windows : setx OLLAMA_API_KEY=your_api_key_here (if you want the windows to remember or use set command if you just need temporarily)
7. run the program i.e pmchat-windows-amd64.exe (windows) or ./pmchat-linux-amd64
